视图文件¶

视图文件的扩展名为.tpl 或 .php, 如index.tpl 或 index.php

注解: 如果是.tpl文件, 那么视图的基本语法请参考模板语法部分的说明.

视图文件分为3种类型. 类型1: 元素文件, 所在目录为

根目录
|-- app
|-- layout
`-- element #全局元素文件所在目录

如果你的元素文件仅提供给当前模块使用, 那么该元素文件所在目录还可以在该模块目录下, 如user模块中的pageurl.tpl元素文件:

根目录
|-- app
|   |-- user
|   |   |-- element
|   |   |   `-- pageurl.tpl #此元素文件仅提供给user模块使用.
|   |   |-- template
|   `-- ..
|-- ....
`-- element

类型2: 布局文件, 所在目录

根目录
|-- layout #全局布局文件所在目录
`-- element
注解: 如果当前模块调用的视图文件, 在当前模块中存在, 且在全局目录中存在, 那么最终成功调用的将会是当前模块中的视图文件.
